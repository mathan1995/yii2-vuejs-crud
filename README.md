<h1>Yii2-Vue.Js-CRUD</h1>
(https://raw.githubusercontent.com/sn8/yii2-vuejs-crud/master/screenshot.png)

Simple CRUD application with Yii2 and Vue.js.

INSTALLATION
-------------
~~~
composer install
npm install
npm run dev
~~~

Edit the file `config/db.php` with real data and then run:
~~~
php yii migrate/up
~~~
